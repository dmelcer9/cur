===

Cur proof assistant, implemented with Turnstile+ framework.

this branch: `popl2020-artifact`
=============================

## install instructions
  - install Racket
  - install Turnstile+
  - To install Cur: `make install` (from repo root)
  - To remove Cur: `make remove` (from repo root)
